🎵 How It Works (Tap to show/hide)

A lightweight, fun way to train groove transitions with randomized BPM ranges. Includes a full-featured Groove Randomizer for practice sessions and a Simple Metronome for everyday use.

Quick Start:

  • Set your desired BPM range (multiples of 5)
  • Enter groove names (one per line)
  • Choose session duration or cycle count
  • Press Space or tap Start

Features:

  • Randomized BPM and groove selection
  • Built-in metronome with accurate timing
  • Rhythmic Count-in (toggle available)
  • Visual beat indicators with subdivisions
  • Dark mode and offline support

Hotkeys:

  • H = Show this help
  • Space = ▶️ Start / ⏹️ Stop
  • P = ⏸️ Pause / ▶️ Resume
  • N = ⏭️ Next Groove
  • ↑ / ↓ = Adjust BPM (±BPM Step)
  • ← / → = Switch Min/Max
